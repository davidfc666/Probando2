Manual Apache

  • Limpiar

    Para comenzar limpiaremos el sistema.

    yum clean all

  • Actualizar

    Actualizaremos el sistema.

    yum -y update

  • Instalar apache

    Instalaremos Apache.

    yum install httpd

  • Instalar firewall

    Instalaremos el firewall.

    yum install firewalld

  • Cambiar fichero de configuración

    Cambiaremos el fichero de configuración de Apache.

    nano /etc/httpd/conf/httpd.conf

    En "AllowOverride" pondremos "ALL".

  • Habilitar Apache

    Habilitaremos Apache.

    systemctl enable httpd

  • Iniciar Apache

    Después iniciaremos Apache.

    systemctl start httpd

  • Comprobar estado

    Comprobaremos el estado de Apache.

    systemctl status httpd

  • Cambiar puertos

    Cambiaremos los puertos del firewall.

    firewall-cmd --permanent --add-port=80/tcp
    firewall-cmd --permanent --add-port=80/udp
    firewall-cmd --permanent --add-port=443/tcp

    Reiniciaremos el firewall.

    firewall-cmd --reload

  • Reiniciar Apache

    Le damos todos los permisos a la carpeta prestashop.

    chmod -R 777 prestashop

  • Reiniciar firewall

    Reiniciaremos el firewall.

    firewall-cmd --reload

  • Crear index de Apache

    nano /var/www/html/index.html y escribir lo que queramos
