Manual WordPress

  • Conectar BD

    Nos conectamos a la base de datos.

    mysql -u root -p

  • Crear BD

    Creamos la base de datos de wordpress.

    CREATE DATABASE wordpress;

  • Crear usuario

    Crearemos un usuario para wordpress.

    CREATE USER usuariowordpress@localhost IDENTIFIED BY 'Grupo2'

  • Privilegios

    Le asignaremos permisos al usuario anterior.

    GRANT ALL PRIVILEGES ON wordpress.* TO usuariowordpress@localhost IDENTIFIED BY 'Grupo2'
    FLUSH PRIVILEGES;

  • Descargar

    Descargaremos la última versión de wordpress.

    wget http://wordpress.org/latest.tar.gz

  • Instalar

    Instalaremos el wordpress descargado anteriormente

    unzip wordpress.zip

  • Permisos

    Daremos permisos a la carpeta.

    chown -R 777 /var/www/html/wordpress
