Manual PrestaShop

  • Conectar BD

    Nos conectamos a la base de datos.

    mysql -u root -p

  • Crear BD

    Creamos la base de datos de prestashop.

    CREATE DATABASE prestashop;

  • Crear usuario

    Crearemos un usuario para prestashop.

    CREATE USER usuariopresta@localhost IDENTIFIED BY 'Grupo2'

  • Privilegios

    Le asignaremos permisos al usuario anterior.

    GRANT ALL PRIVILEGES ON prestashop.* TO usuariopresta@localhost IDENTIFIED BY 'Grupo2'
    FLUSH PRIVILEGES;

  • Descargar

    Descargaremos la última versión de prestashop.

    wget http://download.prestashop.com/download/releases/prestashop_1.7.2.4.zip

  • Instalar

    Instalaremos el prestashop descargado anteriormente

    unzip prestashop_1.6.1.5.zip

  • Permisos

    Daremos permisos a la carpeta.

    chown -R 777 /var/www/html/prestashop
