Manual CentOs7

  • Instalar CentOs

    Crearemos un nuevo contenedor en el servidor con la plantilla de CentOs.

  • Actualizar

    Actualizaremos el sistema.

    yum -y update || yun upgrade

  • Instalar nano

    Instalaremos el editor nano.

    yum install nano

  • Hacerlo predefinido

    Haremos que nano sea nuestro editor predefinido.

    nano /etc/bashrc

    Y en la última línea, escribimos "Export EDITOR ="nano"

  • Reiniciar

    Reiniciaremos todo.

    reboot
