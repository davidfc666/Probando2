Manual PHP-MyAdmin

  • Instalar EPEL

    Instalaremos el repositorio EPEL.

    yum install epel-release

  • Instalar PHP-MyAdmin

    Instalaremos PHP-MyAdmin.

    yum install phpmyadmin

  • Archivo configuración

    Accederemos al archivo de configuración

    nano /etc/httpd/conf.d/phpMyAdmin.conf

    En "Require ip" y "Allow from" pondremos nuestra ip.

  • Reiniciar apache

    Reiniciaremos apache

    systemctl restart httpd.service

  • Acceder

    Accederemos mediante un navegador
