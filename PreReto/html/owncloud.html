Manual OwnCloud

  • Conectar BD

    Nos conectamos a la base de datos.

    mysql -u root -p

  • Crear BD

    Creamos la base de datos de owncloud.

    CREATE DATABASE owncloud;

  • Crear usuario

    Crearemos un usuario para owncloud.

    CREATE USER adminowncloud@localhost IDENTIFIED BY 'adminowncloud'

  • Privilegios

    Le asignaremos permisos al usuario anterior.

    GRANT ALL PRIVILEGES ON owncloud.* TO adminowncloud@localhost IDENTIFIED BY 'adminowncloud'
    FLUSH PRIVILEGES;

  • Descargar

    Descargaremos la última versión de owncloud.

    wget https://download.owncloud.org/community/owncloud-10.0.3.zip

  • Instalar

    Instalaremos el owncloud descargado anteriormente

    unzip owncloud_10.0.3.zip

  • Permisos

    Daremos permisos a la carpeta.

    chown -R 777 /var/www/html/owncloud

  • Configuración

    Añadimos plugins
    Añadimos "XMAS"
    Comprobamos "XMAS"
    Añadimos "Text Editor"
    Comprobamos "Text Editor"
